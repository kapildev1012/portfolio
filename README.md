## ✨ Features

- **3D Hero Section** — Interactive Spline 3D robot rendered in real-time
- **Nike-grade Mobile Menu** — Fullscreen clip-path wipe animation with staggered typography and hover tag reveals
- **Curated Project Gallery** — Pinned GitHub projects (Wellfire Ecommerce, Hotspot SmartStock, Portfolio) with live site links
- **Animated Skills Section** — Rotating tech stack logos with kinetic micro-animations
- **SocialMagnet Dock** — Magnetic floating social dock with glassmorphism effect
- **Smooth Scrolling** — Powered by Lenis for buttery-smooth page navigation
- **Fully Responsive** — Dedicated mobile components with premium mobile-first design

## 🚀 Getting Started

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Installation

```bash
# Clone the repo
git clone https://github.com/kapildev1012/React-Personal-Portfolio-main.git
cd React-Personal-Portfolio-main/portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
